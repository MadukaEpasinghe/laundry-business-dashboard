# 🧺 Laundry Business Performance Dashboard

> 📊 Data-driven insights to improve revenue, customer retention, and operational efficiency

---

## 📌 Overview
This project presents a Power BI dashboard designed to analyze the performance of a laundry business. It provides key insights into revenue trends, customer behavior, and service performance.

---

## 🧠 Business Problem
Small businesses often lack visibility into their performance data, making it difficult to identify growth opportunities and inefficiencies.

---

## 💡 Solution
An interactive dashboard was built to:
- Track revenue trends
- Analyze customer patterns
- Identify high-performing services

---

## 📊 Key Insights
- Company earns more in mid populates cities
- Return on marketing investment is highest at mid populated city
- Company should invest more on marketing in mid populated cities 

---

## 🛠 Tools Used
- Power BI
- Data Cleaning & Transformation
